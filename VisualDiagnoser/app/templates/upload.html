upload api

  • url: http://0.0.0.0:9099/upload
  • Method: POST
  • Consumes: multipart/form-data
  • Body
    • lastname
      • type: string
      • description: Last name of person to create.
    • firstname
      • type: string
      • description: First name of person to create.
    • results
      • type: string
      • description: The results converted to string.
    • description
      • type: string
      • description: Some description.
    • image
      • type: file
      • description: The image to upload.
  • Response
    • 200
      • description: successful upload.
      • Returns a json object of the schema.
